# python-flask-show-request-route

## Setup

### Requirements

Install python3.

### Install libs

```bash
pip3 install -r requirements.txt
```

## Usage

```
python3 ./src/main.py
```

## With heroku

Activate heroku cli.

```
NAME_YOUR_HEROKU_APP=asukiaaa-show-route
heroku apps:create $NAME_YORU_HEROKU_APP

```
